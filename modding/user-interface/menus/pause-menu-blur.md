# 暂停菜单模糊

在“暂停”菜单的背景中可以看到的模糊由几个ConVars控制

```text
// Default Values
mat_blur_r 0.7
mat_blur_g 0.7
mat_blur_b 0.7
```

它们使用红、绿和蓝色来控制中模糊的颜色效果

![](../../../.gitbook/assets/sp_boomtown_start0015.jpg)

```text
// Default Values
mat_blur_desaturate 0.4
```

这可以控制颜色的饱和度

```text
// Default Values
r_blurmenubg 1
```

该值可打开和关闭背景模糊
