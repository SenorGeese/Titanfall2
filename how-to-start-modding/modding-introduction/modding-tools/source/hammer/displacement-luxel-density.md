# 位移Luxel密度

这不是一个bug。基于位移的大小，一般位移会有一个最大的luxel密度。如果位移很大，你会发现你没法在位移上生成小尺寸的luxel。一个解决办法是将大的位移分解成多个较小的位移，并将它们组合在一起。

请参阅[位移](https://developer.valvesoftware.com/wiki/Displacement)以获取更多关于如何创建，编辑，剪辑和缝合的信息。
